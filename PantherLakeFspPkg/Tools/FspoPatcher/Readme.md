# [Tool Introduction]
Crypto services are used in both FspLoaderPeim and FspSecCoreO. This tool provides
a post build patch capability to make these services in FspLoaderPeim be able to
share with FspSecCoreO.

# [How to Use]

## Options
-f input of the FSP binary containing FSP-O
